# hours-calculator

מחשבון שעות בעברית (RTL) לחישוב כמה זמן נשאר עד היעד.

## איך מריצים
1. פותחים את הקובץ `index.html` בדפדפן.
2. אפשר גם להפעיל שרת מקומי פשוט ולפתוח משם.

## איפה לערוך טקסטים/צבעים
- טקסטים וכותרות: בקובץ `index.html` (ה-HTML בראש הקובץ).
- צבעים ועיצוב: בתוך הבלוק `:root` ב-`index.html` (משתני צבע).
- לוגיקה וחישובים: בתחתית `index.html` בתוך `<script>`.

## פורמטי קלט נתמכים
- שעות עשרוניות: `7.5`
- שעות בפורמט זמן: `7:30`
